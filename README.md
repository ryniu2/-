# 韭菜成长史

一个静态网页小游戏原型：全屏 Canvas 走位生存，随机抽取散户角色，专属技能自动打怪，每 10 级挑战一个股市主题 Boss，最终从小韭菜成长为股市大牛。

## 运行方式

直接用浏览器打开 `index.html` 即可游玩，不需要安装依赖。

## 当前内容

- 23 个角色与专属攻击/技能设定，并绑定 23 张头像。
- WASD / 方向键移动，手机按住屏幕拖动。
- 怪物从四周追来，角色会自动释放不同攻击：剑气环绕、火焰环绕、冰霜发射、子弹扫射、炮弹、光束等。
- 每次升级都会强化当前角色的攻击：环绕体更多、弹幕更多、伤害更高、冷却更短或范围更大。
- 每 2 级弹出一次“三选一”技能树强化，能走伤害、范围、攻速、护盾、回血、暴击、大招流派。
- 加入金币机制：击杀普通怪、精英怪和 Boss 都会获得金币，金币可在技能树选择界面刷新当前三选一。
- 加入技能树分支：燃烧、冰冻、雷电、暴风、骤雨、飞剑，重复选择同一分支会逐层强化。
- 元素命中会有明显动画：燃烧火焰、冰冻结晶、雷电链、暴风旋涡、骤雨落点和状态小图标。
- 加入元素反应：冰冻后雷电伤害翻倍，骤雨叠加冰冻会造成寒雨冻结，燃烧叠加暴风会触发焚风增伤。
- 空格键或右下角“大招”按钮可释放主动爆发，手机也可以直接点按钮。
- Boss 出场会嘲讽，并拥有不同技能：旋转镰刀、小作文弹幕、高位砸盘、爆仓冲撞、熊市寒潮、量化矩阵。
- Boss 血量、伤害、护盾、阶段机制和召唤频率会随关卡递进；击败 Boss 后可获得对应 Boss 技能并永久强化本局。
- 每一关拥有不同市场主题背景，击败 Boss 后会播放过关转场动画进入下一关。
- 普通怪和 Boss 外观做了差异化绘制，包含飞行怪、石头怪、螃蟹怪、冲刺怪、特效怪、恢复怪、尖刺 Boss 等视觉风格。
- 怪物攻击模式扩展：冲刺猛兽会突进，特效妖姬会减速/弹幕/爆闪，回血庄家会给周围怪物恢复。
- 普通怪和精英怪会掉落经验珠、回血包、护盾包、暴击药水和大招充能，靠近玩家会自动吸附拾取。
- 每隔一段时间会出现精英怪，带有光环、更多血量和特殊技能；击杀后掉落更好。
- 游戏结束会显示本局结算，包括存活时间、等级、关卡、击杀、金币、精英击杀、拾取数量和 Boss 技能。
- 加入动态难度压力：关卡、等级、存活时间、击杀数越高，怪物血量、伤害、速度、刷新密度和 Boss 压迫感都会提升。
- 随机普通怪、经验升级、护盾回血、Boss 闯关。
- 每 10 级解锁一关 Boss：韭菜收割机、黑嘴荐股王、高位站岗巨像、杠杆爆仓兽、熊市寒潮龙、量化镰刀主宰。
